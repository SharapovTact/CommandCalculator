# Командная строка(Калькулятор на мониторе)
## Цель: создать устройство, которое генерирует изображение через VGA порт.  
## Функционал:
- Пользователь должен видеть на мониторе операнд 1, оператор, операнд 
2 и результат этой функции. 
- Машина выполняет функцию по действию-подтверждению - Пользователь должен иметь доступ ко всем данным функции до 
действия-подтверждения.
- Шрифт текста на мониторе определён константно.
